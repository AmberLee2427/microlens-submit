Microlensing Data Challenge Submission Guide
===========================================

Welcome to the official documentation for ``microlens-submit``. During the 2018 Data Challenge, teams mailed in spreadsheets and wrote pages of answers by hand. That process was tedious and error-prone. For Data Challenge 2, we provide this tool to automate everything from project setup to the final zip file. Using ``microlens-submit`` is **required** for packaging your results.
